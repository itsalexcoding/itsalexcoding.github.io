Recursion

In programming, recursion is the process by which a function calls itself. 

public long factorial(long n) {
      if (n <= 1) {
        return 1;
      }
      return n * factorial(n - 1);
    }
    /* factorial calculates n! by evaluating n. If n evaluates to a value lesser than or equal to 1, then factorial returns 1. */
    /* If not, then factorical calls itself to evaluate the factorial (n - 1), which also reduces the value of n by 1 as well when return n * factorial(n - 1) is evaluated again. */
    /* factorial recurses until n is lesser than or equal to 1; return n * factorial(n - 1) is evaluated, then returns the evaluation. */
  
    /* 
    5! = 5 * 4!
    4! = 4 * 3!
    3! = 3 * 2!
    2! = 2 * 1!
    1! = 1 * 1! = 1

    The factorial of 1 cannot be more elementary; meaningfully.
    */
