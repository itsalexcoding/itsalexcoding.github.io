Variables and Data Types

Variables

A variable is a named reference to a specific memory location, storing data and subject to manipulated during execution.
To properly utilize a variable, it must declared, then initialized.

Declaration and Initialization

Declaration specifies the data type and name of a variable; initialization either occurs during declaration or some time thereafter, and is the assignment of a value to the name of the variable. 

      int num; // The variable 'num' can store integer values.
      num = 5; // 'num' is storing an integer value of '5' as a field.
      char letter = 'a' // Declaration and initialization may occur on the same line.

Data Type

The data type refers to the size and characteristics of the data stored by a variable.

Primitive Data Type

A primitive data type represents fundamental values, which are pre-defined by Java and are not further divisible (into a more elemental form). Each keyword specifies the size and structural attributes of storable values, indicating the type of the variable.
Due to their simple nature, primitive data types are immutable, are not called by method no methods, and are temporarily stored in stack memory.
There are eight primitive data types in Java:

  • byte, short, int and long represent integer values
  • float and double represent floating points
  • boolean represents true and false values
  • char represents Unicode characters
      /* All integers are signed two's complement.

      float floatNum = 3.4028235E38f; // 32 bits; 6-7 decimal digits
